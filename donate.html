Important Note for Stripe Integration:

For this donation feature to work securely and dynamically, a **backend server** is required to create a Stripe Checkout Session. This static HTML page can only initiate the client-side Stripe.js library. The 'Donate Now' button currently serves as a placeholder until a backend endpoint is configured to provide the `sessionId`.

You will need to:

  1. Set up a server-side endpoint (e.g., `/create-checkout-session`) that uses your **Stripe Secret Key** (from your `.env` file on the server) to create a Stripe Checkout Session based on the `amount` provided.
  2. This server-side endpoint should return the `sessionId` to the client.
  3. In the script below, replace `YOUR_STRIPE_PUBLISHABLE_KEY` with your actual publishable key and modify the `fetch` call to your backend endpoint.
